# pdfReader

Простая и эффективная консольная "читалка" для PDF-файлов на C#. Программа извлекает текст из PDF, аккуратно склеивает разорванные предложения, сохраняет структуру списков и выводит текст на экран с заданной скоростью.

## Особенности

* **Умная склейка текста**: PDF часто разрывает предложения при переходе на новую строку. Эта программа склеивает их обратно в целые абзацы.
* **Сохранение структуры**: Распознает заголовки и пункты списков (1., •, №), не давая им сливаться с основным текстом.
* **Перенос по словам**: Текст автоматически подстраивается под ширину вашего консольного окна, слова никогда не разрезаются пополам.
* **Режим чтения**: Вывод текста происходит с паузами, что позволяет комфортно читать документ прямо из консоли.
* **Массовая обработка**: Можно указать путь к конкретному файлу или к целой папке с PDF-документами.
